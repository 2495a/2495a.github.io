webpack的配置

阅读全文

1. 創建一個名為webpack的文件夾

npm init -y 初始化文件夾 自動生成package.json文件

2. 下載webpack 以及 webpack-cli

npm install webpack webpack-cli -D 這裡-D下载到开发依赖项中

3. 在webpack文件夾下創建如下目錄

3.1 在webpack.config.js中配置webpack打包的入口和出口

3.2 配置脚本命令

npm install webpack-dev-server -D

3.3 生成创建html入口文件

npm install html-webpack-plugin -D

3.3 配置webpack开发服务

3.4 生成创建html入口文件

npm install html-webpack-plugin -D

3.5 编译css文件

npm install style-loader css-loader postcss-loader autoprefixer -D

npm install less-loader -D

3.6 提取打包css

npm install mini-css-extract-plugin -D 此插件为webpack4.x插件

3.7 处理js

npm install @babel/core @babel/preset-env @babel/core babel @babel-preset-env

npm install babel-loader

3.8 处理图片、字体与媒体文件

npm install url-loader

3.9 处理html中的img图

npm install html-withimg-loader -D

3.10 配置优化项

npm install optimize-css-assets-webpack-plugin uglifyjs-webpack-plugin -D

以上是webpack简单的基础配置
